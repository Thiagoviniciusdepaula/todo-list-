🧠 To-Do List

Projeto desenvolvido em Python com o objetivo de praticar conceitos fundamentais da linguagem por meio da construção de uma aplicação de lista de tarefas executada no terminal. A aplicação permite ao usuário adicionar, listar e remover tarefas de forma interativa, utilizando estruturas de dados como listas para armazenamento em memória. O projeto foi organizado com funções para promover modularização e melhor leitura do código, além de aplicar estruturas de controle como loops para manter a execução contínua e interação direta com o usuário
